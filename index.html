What is CramBot on the first place

CramBot is a discord utility bot. The bot is inpired in the youtuber MrCramYT

This bot allows you to do some stuff. Here i will post some examples. The default prefix is $

Commands

 

Administration:

 

Name of the Command Example of use Bot answer
Ban $ban @unzor The bot will ban the user specified, in this case @unzor
Embed Creator (not published yet) $embedcreator "Embed title","Description","Image"(optional) The bot will send an embed with the respective strings specified in the last command
Kick $kick @unzor The bot will kick the user specified, in this case @unzor
Mute (not published yet) $mute @unzor The bot will mute the user specified, in this case @unzor
Queue Peek $queuepeek The RPC of the bot will turn into the 2b2t queue
Say $say hey! The bot will say "hey!" (in this case)
SetRPC $setrpc hey! The bot's RPC will say "hey!" (in this case)
Shutdown $shutdown The bot will turn off
